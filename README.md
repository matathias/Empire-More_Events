# Empire Refactored: More Events

A straight-forward mod that adds many new events to [Empire Refactored](https://github.com/matathias/Empire-1_6-Continued), ranging from crises to pure boons; from rare and powerful events that target the entire Empire, to small and benign events that simply add some flavor to a playthrough.

With a larger pool of events to pull from, variety will feel greater than ever before.
